# fastcdn package

快速插入公共库中的静态资源CDN文件
Help you insert static resource CDN files

插件指令：fastcdn:add
command：fastcdn:add

搜索类库名，然后选择对应文件，插入到光标位置
Search library name, and select the file you want. Insert the location of the cursor.
